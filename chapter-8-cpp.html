8  KIỂU DỮ LIỆU CẤU TRÚC

Cung cấp cơ chế cho phép khai báo các kiểu dữ liệu mới để giải quyết theo yêu cầu của bài toán dựa vào những kiểu dữ liệu cơ bản được cài đặt sẵn trong ngôn ngữ lập trình.

8.1 LÝ THUYẾT

Kiểu dữ liệu cấu trúc (struct) là gom nhóm các phần tử có thể không cùng kiểu dữ liệu.

Khái báo kiểu dữ liệu cấu trúc

Trong C++, ta có thể khai báo một struct một cách đơn giản.

struct ⟨Tên cấu trúc⟩ {
    ⟨Các dữ liệu thành phần⟩
};

Việc sử dụng typedef như trong C vẫn được hỗ trợ nhưng không còn cần thiết cho việc khai báo biến cấu trúc.

// Cách làm cũ hơn, vẫn hợp lệ trong C++
typedef struct {
    ⟨Các dữ liệu thành phần⟩
} ⟨Tên kiểu dữ liệu cấu trúc⟩;

Kiểu dữ liệu về ngày DATE gồm các thành phần:

  • Thứ (thu): chuỗi ký tự.
  • Ngày (ngay): số nguyên không dấu 1 byte.
  • Tháng (thang): số nguyên không dấu 1 byte.
  • Năm (nam): số nguyên 2 bytes.

Ta định nghĩa cấu trúc DATE

#include <string>

struct DATE {
    std::string thu;
    unsigned char ngay;
    unsigned char thang;
    short nam;
};

Khai báo biến cấu trúc

Khi ta định nghĩa kiểu dữ liệu tức là ta có một kiểu dữ liệu mới, muốn sử dụng ta phải khai báo biến. Cú pháp khai báo kiểu dữ liệu cũng giống như cách khai báo của các kiểu dữ liệu chuẩn.

⟨Tên cấu trúc⟩ ⟨tên biến⟩;

Khai báo biến x có kiểu cấu trúc DATE

DATE x;

Lưu ý các thành phần của một cấu trúc cũng có thể là kiểu dữ liệu cấu trúc.

Định nghĩa kiểu dữ liệu của học sinh HOCSINH gồm:

  • Mã số học sinh (MSHS): chuỗi ký tự.
  • Họ tên (hoten): chuỗi ký tự.
  • Ngày tháng năm sinh (ngaysinh): kiểu DATE.
  • Địa chỉ (diachi): chuỗi ký tự.
  • Giới tính (phai): chuỗi ký tự.
  • Điểm trung bình (diemtb): số thực.

Ta định nghĩa kiểu HOCSINH như sau:

#include <string>

// Giả định struct DATE đã được định nghĩa ở trên
struct HOCSINH {
    std::string MSHS;
    std::string hoten;
    DATE ngaysinh;
    std::string diachi;
    std::string phai;
    float diemtb;
};

Truy xuất thành phần

⟨Tên biến cấu trúc⟩.⟨Tên thành phần⟩

Truy xuất thành phần ngay của biến x

DATE x; // khai báo biến x kiểu DATE
x.ngay = 5; // gán ngay bằng 5

Viết chương trình nhập vào toạ độ hai điểm trong mặt phẳng và tính tổng hai toạ độ này.

#include <iostream>

// Khai báo một kiểu dữ liệu DIEM gồm tọa độ x và y
struct DIEM {
    int x;
    int y;
};

void Nhap(DIEM &d) {
    std::cout << "\nNhap vao toa do diem\n";
    std::cout << "Hoanh do: ";
    std::cin >> d.x;
    std::cout << "Tung do: ";
    std::cin >> d.y;
}

void Xuat(DIEM d) {
    std::cout << "\nToa do diem: (" << d.x << ", " << d.y << ")";
}

DIEM Tong(DIEM d1, DIEM d2) {
    DIEM temp;
    temp.x = d1.x + d2.x;
    temp.y = d1.y + d2.y;
    return temp;
}

int main() {
    DIEM A, B, AB; // khai bao 3 diem A, B, AB
    Nhap(A);
    Xuat(A);
    Nhap(B);
    Xuat(B);
    std::cout << "\nTong cua hai diem vua nhap la: ";
    AB = Tong(A, B);
    Xuat(AB);
    std::cout << std::endl;
    return 0;
}

Mảng cấu trúc

  • Cách khai báo tương tự như mảng một chiều hay ma trận. Trong C++ hiện đại, nên sử dụng std::vector để thay thế cho mảng C-style.
  • Cách truy cập phần tử trong mảng cũng như truy cập trên mảng một chiều hay ma trận. Nhưng do từng phần tử có kiểu cấu trúc nên phải chỉ định rõ cần lấy thành phần nào, tức là phải truy cập đến thành phần cuối cùng có kiểu là dữ liệu cơ bản (xem lại bảng các kiểu dữ liệu cơ bản).

Kỹ thuật viết chương trình có mảng cấu trúc

Do kiểu dữ liệu có cấu trúc thường chứa rất nhiều thành phần nên khi viết chương trình loại này ta cần lưu ý:

  • Xây dựng hàm xử lý cho một kiểu cấu trúc.
  • Muốn xử lý cho mảng cấu trúc, ta gọi lại hàm xử lý cho một kiểu cấu trúc đã được xây dựng bằng cách dùng vòng lặp.

Cho một lớp học gồm n học sinh. Thông tin của một học sinh được mô tả trong ví dụ trước. Hãy viết chương trình nhập và xuất danh sách học sinh sau đó đếm xem có bao nhiêu học sinh được lên lớp (Điều kiện được lên lớp là điểm trung bình \geq5.0). Cách làm:

  • Trước hết ta phải xây dựng hàm nhập và xuất cho 1 học sinh.
  • Xây dựng hàm nhập và xuất ngày tháng năm (Kiểu dữ liệu DATE).
  • Sau đó mới xây dựng hàm nhập và xuất cho danh sách học sinh.
#include <iostream>
#include <vector>
#include <string>
#include <limits> // Để sử dụng numeric_limits

struct DATE {
    unsigned char ngay;
    unsigned char thang;
    int nam;
};

struct HOCSINH {
    std::string MSHS;
    std::string hoten;
    DATE ngaysinh;
    std::string diachi;
    std::string phai;
    float diemtb;
};

void NhapNamSinh(DATE &d);
void XuatNamSinh(DATE d);
void Nhap1HS(HOCSINH &hs);
void Xuat1HS(HOCSINH hs);
void NhapDSHS(std::vector<HOCSINH> &lh, int &n);
void XuatDSHS(const std::vector<HOCSINH> &lh, int n);
int DemHSLenLop(const std::vector<HOCSINH> &lh, int n);

int main() {
    std::vector<HOCSINH> lh; // Sử dụng vector để quản lý danh sách
    int n, sohsdau;
    NhapDSHS(lh, n);
    XuatDSHS(lh, n);
    sohsdau = DemHSLenLop(lh, n);
    std::cout << "\nSo luong hoc sinh duoc len lop la: " << sohsdau << std::endl;
    return 0;
}

void NhapNamSinh(DATE &d) {
    std::cout << "Nhap vao ngay: ";
    std::cin >> d.ngay;
    std::cout << "Nhap vao thang: ";
    std::cin >> d.thang;
    std::cout << "Nhap vao nam: ";
    std::cin >> d.nam;
}

void XuatNamSinh(DATE d) {
    // std::cout << d.ngay << " / " << d.thang << " / " << d.nam;
    printf("%02u / %02u / %4d", d.ngay, d.thang, d.nam); // printf dễ định dạng hơn
}

void Nhap1HS(HOCSINH &hs) {
    // Xóa bộ đệm đầu vào trước khi đọc chuỗi
    std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');

    std::cout << "Nhap ma so hoc sinh: ";
    getline(std::cin, hs.MSHS);

    std::cout << "Nhap ho ten hoc sinh: ";
    getline(std::cin, hs.hoten);

    std::cout << "Nhap ngay thang nam sinh:\n";
    NhapNamSinh(hs.ngaysinh);
    
    std::cin.ignore(std::numeric_limits<std::streamsize>::max(), '\n');

    std::cout << "Nhap vao dia chi: ";
    getline(std::cin, hs.diachi);

    std::cout << "Phai: ";
    getline(std::cin, hs.phai);

    std::cout << "Nhap vao diem trung binh: ";
    std::cin >> hs.diemtb;
}

void NhapDSHS(std::vector<HOCSINH> &lh, int &n) {
    std::cout << "Nhap vao so luong hoc sinh: ";
    std::cin >> n;
    lh.resize(n); // Thay đổi kích thước vector
    for (int i = 0; i < n; i++) {
        std::cout << "\nNhap vao thong tin cua hoc sinh thu " << i + 1 << ":\n";
        Nhap1HS(lh[i]); // Gọi hàm nhập thông tin 1 học sinh
    }
}

void Xuat1HS(HOCSINH hs) {
    std::cout << "\nMa so hoc sinh: " << hs.MSHS;
    std::cout << "\nHo ten hoc sinh: " << hs.hoten;
    std::cout << "\nNgay thang nam sinh: ";
    XuatNamSinh(hs.ngaysinh);
    std::cout << "\nDia chi: " << hs.diachi;
    std::cout << "\nPhai: " << hs.phai;
    printf("\nDiem trung binh: %.2f", hs.diemtb);
}

void XuatDSHS(const std::vector<HOCSINH> &lh, int n) {
    for (int i = 0; i < n; i++) {
        std::cout << "\n\nThong tin hoc sinh thu " << i + 1 << ":";
        Xuat1HS(lh[i]); // Gọi hàm xuất thông tin 1 học sinh
    }
}

int DemHSLenLop(const std::vector<HOCSINH> &lh, int n) {
    int d = 0;
    for (int i = 0; i < n; i++)
        if (lh[i].diemtb >= 5.0)
            d++;
    return d;
}

Cho một mảng các phân số (PHANSO). Hãy viết chương trình nhập và xuất danh sách các phân số sau đó tìm phân số có giá trị lớn nhất, tổng và tích các phân số và nghịch đảo giá trị các phân số trong mảng. Cách làm:

  • Trước hết ta phải xây dựng hàm nhập và xuất cho 1 phân số.
  • Xây dựng hàm tính tổng, hiệu, tích, thương, rút gọn, so sánh và nghịch đảo cho 2 phân số.
  • Sau đó mới xây dựng hàm nhập, xuất, tính tổng, tích cho mảng các phân số.
#include <iostream>
#include <vector>
#include <numeric> // For std::gcd in C++17
#include <cmath>   // For abs

// Hàm tìm UCLN, có thể dùng std::gcd từ C++17
int USCLN(int a, int b) {
    a = abs(a);
    b = abs(b);
    while (a != b) {
        if (a > b) a = a - b;
        else b = b - a;
    }
    return a;
}

struct PHANSO {
    int tu, mau;
};

PHANSO RutGon(PHANSO ps) {
    if (ps.tu == 0) {
        ps.mau = 1;
        return ps;
    }
    int us = USCLN(ps.tu, ps.mau);
    ps.tu /= us;
    ps.mau /= us;
    if (ps.mau < 0) { // Đưa dấu trừ lên tử
        ps.tu = -ps.tu;
        ps.mau = -ps.mau;
    }
    return ps;
}

void NhapPS(PHANSO &ps) {
    do {
        std::cout << "Nhap tu so: ";
        std::cin >> ps.tu;
        std::cout << "Nhap mau so: ";
        std::cin >> ps.mau;
        if (ps.mau == 0)
            std::cout << "\nMau so khong duoc bang 0, nhap lai phan so\n";
        else
            break;
    } while (true);
    ps = RutGon(ps);
}

void XuatPS(PHANSO ps) {
    if (ps.mau == 1 || ps.tu == 0) {
        std::cout << ps.tu;
    } else {
        std::cout << ps.tu << "/" << ps.mau;
    }
}

void NhapMangPS(std::vector<PHANSO> &dsps) {
    int n;
    std::cout << "\nNhap so luong phan so: ";
    std::cin >> n;
    dsps.resize(n);
    for (int i = 0; i < n; i++) {
        std::cout << "\nNhap vao phan so thu " << i + 1 << ":\n";
        NhapPS(dsps[i]);
    }
}

void XuatMangPS(const std::vector<PHANSO> &dsps) {
    for (const auto &ps : dsps) {
        XuatPS(ps);
        std::cout << "\t";
    }
}

PHANSO NghichDao(PHANSO ps) {
    PHANSO kq;
    kq.tu = ps.mau;
    kq.mau = ps.tu;
    return kq;
}

PHANSO Nhan(PHANSO ps1, PHANSO ps2) {
    PHANSO kq;
    kq.tu = ps1.tu * ps2.tu;
    kq.mau = ps1.mau * ps2.mau;
    return RutGon(kq);
}

PHANSO Cong(PHANSO ps1, PHANSO ps2) {
    PHANSO kq;
    kq.tu = ps1.tu * ps2.mau + ps1.mau * ps2.tu;
    kq.mau = ps1.mau * ps2.mau;
    return RutGon(kq);
}

// Trả về 1 nếu ps1 > ps2, -1 nếu ps1 < ps2, 0 nếu bằng nhau
int SoSanh(PHANSO ps1, PHANSO ps2) {
    double val1 = static_cast<double>(ps1.tu) / ps1.mau;
    double val2 = static_cast<double>(ps2.tu) / ps2.mau;
    if (val1 > val2) return 1;
    if (val1 < val2) return -1;
    return 0;
}

PHANSO TimMax(const std::vector<PHANSO> &dsps) {
    PHANSO max = dsps[0];
    for (size_t i = 1; i < dsps.size(); i++) {
        if (SoSanh(dsps[i], max) == 1) {
            max = dsps[i];
        }
    }
    return max;
}

PHANSO TongCacPS(const std::vector<PHANSO> &dsps) {
    PHANSO s = {0, 1};
    for (const auto &ps : dsps) {
        s = Cong(s, ps);
    }
    return s;
}

PHANSO TichCacPS(const std::vector<PHANSO> &dsps) {
    PHANSO p = {1, 1};
    for (const auto &ps : dsps) {
        p = Nhan(p, ps);
    }
    return p;
}

void NghichDaoCacPS(std::vector<PHANSO> &dsps) {
    for (auto &ps : dsps) {
        ps = NghichDao(ps);
    }
}

int main() {
    std::vector<PHANSO> a;
    NhapMangPS(a);
    
    std::cout << "\nMang cac phan so vua nhap: ";
    XuatMangPS(a);
    
    PHANSO max = TimMax(a);
    std::cout << "\nPhan so co gia tri lon nhat: ";
    XuatPS(max);
    
    PHANSO s = TongCacPS(a);
    std::cout << "\nTong gia tri cac phan so co trong mang: ";
    XuatPS(s);
    
    PHANSO p = TichCacPS(a);
    std::cout << "\nTich gia tri cac phan so co trong mang: ";
    XuatPS(p);
    
    NghichDaoCacPS(a);
    std::cout << "\nMang phan so sau khi nghich dao cac phan tu: ";
    XuatMangPS(a);
    
    std::cout << std::endl;
    return 0;
}

8.2 BÀI TẬP

Bài tập cơ bản

  1. Viết chương trình sử dụng con trỏ cấu trúc để hiển thị giờ, phút, giây ra màn hình, và tính khoảng cách giữa 2 mốc thời gian.
  2. Viết chương trình sử dụng con trỏ cấu trúc thể hiện ngày, tháng, năm ra màn hình, và tính khoảng cách giữa 2 ngày.
  3. Viết chương trình khai báo kiểu dữ liệu thể hiện một số phức. Sử dụng kiểu này để viết hàm tính tổng, hiệu, tích của hai số phức.
  4. Viết chương trình khai báo kiểu dữ liệu để biểu diễn một phân số. Hãy viết hàm thực hiện những công việc sau:
    1. Tính tổng, hiệu, tích, thương hai phân số.
    2. Rút gọn phân số.
    3. Qui đồng hai phân số.
    4. So sánh hai phân số.
  5. Viết chương trình khai báo kiểu dữ liệu để biểu diễn một hỗn số. Hãy viết hàm thực hiện những công việc sau:
    1. Đổi hỗn số sang phân số
    2. Tính tổng, tích hai hỗn số
  6. Viết chương trình khai báo kiểu dữ liệu để biểu diễn một điểm trong hệ tọa độ 0xy. Hãy viết hàm thực hiện các công việc sau:
    1. Tìm những điểm đối xứng của nó qua tung độ, hoành độ, toạ độ tâm.
    2. Hãy tính tổng, hiệu, tích của hai điểm trong mặt phẳng toạ độ 0xy.
    3. Tính khoảng cách giữa hai điểm.
  7. Cho một hình trụ có các thông tin sau: BanKinh (bán kính hình trụ kiểu số thực), ChieuCao (chiều cao hình trụ kiểu số thực). Hãy thực hiện các công việc sau.
    1. Nhập dữ liệu cho hình trụ trên.
    2. Tính diện tích xung quanh, diện tích toàn phần, thể tích hình trụ.

Bài tập luyện tập và nâng cao

  1. Viết chương trình tạo một mảng các số phức. Hãy viết hàm tính tổng, tích các số phức có trong mảng.
  2. Viết chương trình tạo một mảng các phân số. Hãy viết hàm thực hiện các công việc sau:
    1. Tính tổng tất cả các phân số (kết quả dưới dạng phân số tối giản)
    2. Tìm phân số lớn nhất, phân số nhỏ nhất.
    3. Sắp xếp mảng tăng dần.
  3. Viết chương trình khai báo kiểu dữ liệu STACK (cơ chế LIFO). Viết hàm làm những công việc sau:
    1. Kiểm tra STACK rỗng
    2. Kiểm tra STACK đầy
    3. Thêm phần tử vào STACK
    4. Lấy phần tử ra khỏi STACK
  4. Tổ chức dữ liệu để quản lí sinh viên bằng cấu trúc mẫu tin trong một mảng N phần tử, mỗi phần tử có cấu trúc như sau:
    • Mã sinh viên.
    • Tên.
    • Năm sinh.
    • Điểm toán, lý, hoá, điểm trung bình. Viết chương trình thực hiện những công việc sau:
    1. Nhập danh sách các sinh viên cho một lớp học.
    2. Xuất danh sách sinh viên ra màn hình.
    3. Tìm sinh viên có điểm trung bình cao nhất.
    4. Sắp xếp danh sách lớp theo thứ tự tăng dần của điểm trung bình.
    5. Sắp xếp danh sách lớp theo thứ tự giảm dần của điểm toán.

    7. Tìm sinh viên có tuổi lớn nhất.
    8. Nhập vào tên của một sinh viên. Tìm và in ra các thông tin liên quan đến sinh viên đó (nếu có).
  5. Tổ chức dữ liệu quản lí danh mục các bộ phim VIDEO, các thông tin liên quan đến bộ phim này như sau:

8.3 TÓM TẮT

Kiểu dữ liệu có cấu trúc cho phép ta định nghĩa những kiểu dữ liệu bất kỳ trên cơ sở là những kiểu dữ liệu cơ bản có sẵn trong ngôn ngữ lập trình. Khi xây dựng xong kiểu dữ liệu mới ta phải định nghĩa những thao tác cho kiểu dữ liệu đó. Những kiểu dữ liệu tự định nghĩa này thông thường có rất nhiều thành phần, mỗi thành phần cũng có thể là một kiểu dữ liệu tự định nghĩa, vấn đề là ta chọn kiểu dữ liệu cơ bản nào để xây dựng nên chúng sao cho phù hợp về mặt kiểu dữ liệu và phù hợp về kích thước lưu trữ (vừa đủ). Cách sử dụng những kiểu dữ liệu tự định nghĩa cũng giống như các kiểu dữ liệu cơ bản. Muốn sử dụng phải khai báo biến, khi truy cập các thành phần phải truy cập theo quy ước. Khi làm việc với mảng các kiểu dữ liệu có cấu trúc, ta nên xử lý cho từng thành phần cấu trúc rồi mới xử lý cho mảng cấu trúc bằng cách dùng vòng lặp. Việc sử dụng std::vector và std::string trong C++ giúp quản lý bộ nhớ và chuỗi ký tự an toàn và hiệu quả hơn.
