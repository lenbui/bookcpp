10  KIỂU DỮ LIỆU CON TRỎ

Chương này cung cấp một kiểu dữ liệu mới dùng để quản lý địa chỉ của các biến, cách cấp phát bộ nhớ động và các thao tác liên quan trong C và C++.

10.1 LÝ THUYẾT

Khái niệm

  • Bộ nhớ máy tính bao gồm các ô nhớ, mỗi ô nhớ có kích thước 1 byte.
  • Mỗi ô nhớ có địa chỉ duy nhất là một số nguyên không âm.
  • Mỗi biến sẽ được gắn với một số ô nhớ liên tục.
  • Địa chỉ của biến là địa chỉ của ô nhớ đầu tiên.
  • Địa chỉ NULL (trong C) hoặc nullptr (trong C++) là địa chỉ 0, biểu thị con trỏ không trỏ đến vùng nhớ hợp lệ.

Biến lưu địa chỉ là biến con trỏ.

Khai báo con trỏ

// C và C++
⟨kiểu dữ liệu⟩ *⟨tên biến con trỏ⟩;

Khai báo biến con trỏ và gán địa chỉ

// C++
#include <cstddef> // cho nullptr_t

char  *p1;
int   *p2;
float *p3;

p1 = (char*)0x0010; // Cần ép kiểu tường minh
p2 = (int*)0x0030;
p3 = nullptr;       // Sử dụng nullptr trong C++ hiện đại

Toán tử lấy địa chỉ

// C và C++
&⟨tên biến⟩

Lấy địa chỉ của một biến và lưu lại địa chỉ này vào biến con trỏ

// C++
char c;
int  n;
char *p1;
int  *p2;

p1 = &c;
p2 = &n;

Toán tử truy xuất đến ô nhớ con trỏ quản lý

// C và C++
*⟨tên biến con trỏ⟩

Truy xuất đến biến do biến con trỏ quản lý

// C++
int n;
int *p;
p = &n;
*p = 5; // Tương đương với n = 5;

Thao tác với biến con trỏ

// C++
#include <iostream>

int main() {
    int a = 5;
    int *pa;
    pa = &a;

    std::cout << "Giá trị của a: " << a << std::endl;         // In giá trị của a
    std::cout << "Địa chỉ của a (giá trị của pa): " << pa << std::endl;  // In địa chỉ của a
    std::cout << "Giá trị tại địa chỉ pa trỏ tới: " << *pa << std::endl; // In giá trị của a
    std::cout << "Địa chỉ của chính con trỏ pa: " << &pa << std::endl;   // In địa chỉ của biến con trỏ pa

    return 0;
}

Con trỏ nhiều cấp

Biến con trỏ cũng có địa chỉ của riêng nó.

  • Biến con trỏ cấp 1 là con trỏ quản lý địa chỉ của biến thông thường.
  • Biến con trỏ cấp k quản lý địa chỉ của con trỏ cấp k-1.

Con trỏ cấp 1 p

// C và C++
int *p;

Con trỏ cấp 2 q

// C và C++
int **q;

Con trỏ cấp 3 r

// C và C++
int ***r;

Thao tác với biến con trỏ nhiều cấp

// C++
#include <iostream>

int main() {
    int a = 12;
    int *p = &a;
    int **q = &p;

    std::cout << "a = " << a << std::endl;
    std::cout << "*p = " << *p << std::endl;
    std::cout << "**q = " << **q << std::endl; // Truy xuất giá trị của a thông qua q
    
    return 0;
}

Con trỏ và tham số

Hàm hoanvi sử dụng hai tham số x,y là con trỏ (kiểu C).

// C
void hoanvi(int *x, int *y) 
{ 
    int t = *x; 
    *x = *y; 
    *y = t; 
} 

void main() 
{ 
    int a = 4, b = 7; 
    hoanvi(&a, &b); 
    printf("a = %d, b = %d", a, b); 
}

Hàm hoanvi sử dụng tham chiếu (kiểu C++).

// C++
#include <iostream>

void hoanvi(int &x, int &y) {
    int t = x;
    x = y;
    y = t;
}

int main() {
    int a = 4, b = 7;
    hoanvi(a, b);
    std::cout << "a = " << a << ", b = " << b << std::endl;
    return 0;
}

Cả hai cách đều hoán đổi giá trị của a và b. Tuy nhiên, cách dùng tham chiếu trong C++ thường an toàn và dễ đọc hơn.

Các phép toán trên con trỏ

Toán tử cộng/trừ số nguyên
// C và C++
⟨con trỏ⟩ + ⟨số nguyên⟩
⟨con trỏ⟩ - ⟨số nguyên⟩

Được hiểu là thực hiện phép tính số học ⟨địa chỉ⟩ ± ⟨số nguyên⟩ * sizeof(⟨kiểu dữ liệu⟩) và kết quả trả về cũng là một con trỏ cùng kiểu.

// C++
int a[10];
int *p = a; // p trỏ tới a[0]
p = p + 2;  // p bây giờ trỏ tới a[2]

int *q = &a[5];
q = q - 3; // q bây giờ trỏ tới a[2]

p và q lúc này cùng trỏ tới phần tử a[2].

Toán tử trừ con trỏ
// C và C++
⟨con trỏ 1⟩ - ⟨con trỏ 2⟩

⟨con trỏ 1⟩ và ⟨con trỏ 2⟩ phải cùng kiểu. Kết quả trả về là một số nguyên cho biết khoảng cách (số lượng phần tử) giữa hai con trỏ.

// C++
int a[100];
int *p = &a[50];
int *q = &a[10];
int d = p - q; // d sẽ có giá trị là 40

Giá trị của d là 50 - 10 = 40.

Con trỏ và mảng 1 chiều tĩnh

Tên của mảng chính là một hằng con trỏ, trỏ tới địa chỉ của phần tử đầu tiên.

Khai báo biến mảng a

// C và C++
int a[100];

Khai báo biến con trỏ p,q

// C và C++
int *p,*q;

Gán địa chỉ cho p,q

// C và C++
p = a;
q = &a[0];

p và q sẽ có cùng giá trị địa chỉ. Do đó, a[i], *(a + i), *(p + i) và p[i] đều tương đương nhau.

Con trỏ và cấu trúc

Truy xuất thành phần của biến cấu trúc qua biến con trỏ

// C và C++
(*⟨biến con trỏ⟩).⟨tên thành phần⟩
⟨biến con trỏ⟩->⟨tên thành phần⟩
// C++
#include <iostream>

struct PhanSo {
    int tu, mau;
};

int main() {
    PhanSo a;
    PhanSo *p = &a;
    
    p->tu = 3;      // Cách dùng phổ biến
    (*p).mau = 4;
    
    std::cout << p->tu << "/" << p->mau << std::endl;
    return 0;
}

Con trỏ hàm

  • Con trỏ hàm dùng để quản lý địa chỉ của một hàm.
  • Có thể gọi hàm thực thi thông qua con trỏ hàm, cho phép truyền hàm như một tham số.

Khai báo con trỏ hàm

// C và C++
⟨kiểu trả về⟩ (*⟨biến con trỏ hàm⟩)(⟨danh sách kiểu tham số⟩);
// C++
int (*pf1)(int);
int (*pf2)(double, double);
  • pf1 quản lý địa chỉ của hàm có một tham số int và kiểu trả về int.
  • pf2 quản lý địa chỉ của hàm có hai tham số double và kiểu trả về int.

Gọi thực thi con trỏ hàm

// C và C++
⟨biến con trỏ hàm⟩(⟨danh sách đối số⟩);

Các thao tác trên con trỏ hàm

// C++
#include <iostream>

int Cong(int a, int b) {
    return a + b;
}

int Tru(int a, int b) {
    return a - b;
}

int main() {
    int s;
    int (*f)(int, int); // Khai báo con trỏ hàm

    f = Cong; // f trỏ đến hàm Cong
    s = f(5, 3); // Gọi hàm Cong qua con trỏ f
    std::cout << "5 + 3 = " << s << std::endl;

    f = Tru; // f trỏ đến hàm Tru
    s = f(5, 3); // Gọi hàm Tru qua con trỏ f
    std::cout << "5 - 3 = " << s << std::endl;
    
    return 0;
}

Cấp phát bộ nhớ động

Trong C

Sử dụng các hàm trong thư viện <cstdlib> (hoặc <malloc.h> trong C cũ).

  • Xin cấp phát vùng nhớ:

    void* malloc(size_t size);

  • Xin cấp phát vùng nhớ theo khối và khởi tạo bằng 0:

    void* calloc(size_t num, size_t size);

  • Thu hồi bộ nhớ:

    void free(void* ptr);
// C
#include <stdio.h>
#include <stdlib.h>

int *p = (int*)malloc(sizeof(int));
int *q = (int*)calloc(10, sizeof(int)); // Cấp phát mảng 10 số nguyên

if (p != NULL) {
    *p = 100;
}

free(p);
free(q);
Trong C++

C++ bổ sung thêm các toán tử quản lý bộ nhớ an toàn hơn về kiểu dữ liệu.

  • Cấp phát một đối tượng:

    new ⟨kiểu dữ liệu⟩;

  • Cấp phát một mảng đối tượng:

    new ⟨kiểu dữ liệu⟩[⟨số phần tử⟩];

  • Thu hồi một đối tượng:

    delete ⟨biến con trỏ⟩;

  • Thu hồi một mảng đối tượng:

    delete[] ⟨biến con trỏ⟩;

Biến p chứa địa chỉ của một vùng nhớ cho một số nguyên, biến q chứa địa chỉ của một vùng nhớ cho một dãy 10 số nguyên.

// C++
int *p = new int;
int *q = new int[10];

*p = 50;
q[0] = 1;

delete p;     // Thu hồi vùng nhớ của p
delete[] q;   // Thu hồi vùng nhớ của mảng q

Lưu ý quan trọng: Vùng nhớ cấp phát bằng new phải được giải phóng bằng delete. Vùng nhớ cấp phát bằng new[] phải được giải phóng bằng delete[]. Trộn lẫn chúng sẽ gây ra lỗi không xác định (Undefined Behavior).

10.2 BÀI TẬP

Bài tập cơ bản

  1. Viết hàm hoán vị giá trị của hai biến số nguyên bằng cách sử dụng con trỏ.
  2. Viết chương trình yêu cầu người dùng nhập vào một chuỗi ký tự, sau đó sử dụng con trỏ để đếm số ký tự trong chuỗi (không dùng strlen).
  3. Viết chương trình khai báo một mảng số nguyên. Sử dụng một con trỏ để duyệt qua mảng và in ra giá trị của từng phần tử.
  4. Viết hàm nhận vào một con trỏ kiểu int và giá trị của con trỏ đó. Hàm sẽ kiểm tra xem con trỏ có phải là nullptr hay không và in ra thông báo tương ứng.

Bài tập cấp phát động

  1. Viết chương trình yêu cầu người dùng nhập vào một số nguyên n. Cấp phát động một mảng n số nguyên. Sau đó, cho người dùng nhập giá trị cho mảng này, tính tổng và in kết quả ra màn hình. Nhớ giải phóng bộ nhớ sau khi dùng xong.
  2. Viết một hàm nhận vào một số nguyên n, cấp phát động một mảng n số nguyên, khởi tạo các phần tử của mảng bằng các số ngẫu nhiên từ 1 đến 100, sau đó trả về con trỏ tới mảng đó. Viết chương trình chính gọi hàm này và in mảng ra màn hình.
  3. Định nghĩa một cấu trúc SinhVien gồm các trường: hoTen (chuỗi ký tự), mssv (số nguyên). Viết chương trình cấp phát động cho một biến kiểu SinhVien, cho người dùng nhập thông tin và in ra màn hình.
  4. Mở rộng bài 7: Viết chương trình yêu cầu nhập số lượng sinh viên n, cấp phát động một mảng n SinhVien, cho phép nhập thông tin cho cả lớp và sau đó in danh sách lớp ra màn hình.

Bài tập nâng cao

  1. Sử dụng con trỏ hàm, viết một hàm SapXep nhận vào một mảng số nguyên, kích thước mảng và một con trỏ hàm so sánh. Viết hai hàm so sánh: SoSanhTangDan và SoSanhGiamDan. Chương trình chính sẽ gọi hàm SapXep để sắp xếp mảng theo cả hai chiều.
  2. Viết chương trình cấp phát động một ma trận m x n (m, n nhập từ bàn phím) bằng cách sử dụng con trỏ cấp 2 (int **matrix). Nhập giá trị cho ma trận và in ma trận ra màn hình. Đừng quên giải phóng bộ nhớ đúng cách.

10.3 TÓM TẮT

Con trỏ là một khái niệm nền tảng, cho phép quản lý bộ nhớ một cách trực tiếp. Trong C++, dù có nhiều công cụ cấp cao hơn như tham chiếu và con trỏ thông minh (smart pointers), việc hiểu rõ con trỏ gốc và cách cấp phát bộ nhớ động với new/delete vẫn cực kỳ quan trọng để làm việc hiệu quả với các cấu trúc dữ liệu phức tạp và tối ưu hóa hiệu năng chương trình. Luôn nhớ quy tắc: cấp phát ở đâu, giải phóng ở đó, và sử dụng đúng cặp new/delete hoặc new[]/delete[] để tránh rò rỉ bộ nhớ.
